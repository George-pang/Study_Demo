语义元素清楚地向浏览器和开发者描述其意义。 非语义元素的例子：div 和 span - 无法提供关于其内容的信息。 语义元素的例子：form、table 以及 img - 清晰地定义其内容。

H5新增header标签，定义页面的头部/页眉部分，个人认为可以看作是页面可见部分(body)的头部区域
nav标签语义，定义页面的导航栏
section标签，定义文档的节。根据 W3C 的 HTML 文献：“节（section）是有主题的内容组，通常具有标题”。
footer标签，定义页面的底部/页脚部分,可看作是body的底部区域
article标签语义:定义文章。
aside标签语义：定义页面的侧边栏
datalist新标签要和linput标签配合使用。
